# Weather App

A simple React weather app built with Vite. Search any city and see the current temperature and wind speed, powered by the free Open-Meteo API (no API key required).

## Run it locally

```bash
npm install
npm run dev
```

Then open the URL shown in your terminal (usually http://localhost:5173).
